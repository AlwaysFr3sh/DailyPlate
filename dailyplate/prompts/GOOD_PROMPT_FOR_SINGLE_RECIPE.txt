Please return a meal recipe in the form of a json object. The should be structured like this: {"<meal_name>" : {"ingredients" : {"<ingredient_1>" : {"<ingredient_proportion> : "<unit_of_measurement">}}, "instructions" : ["<instruction>"]}} where <meal_name>" should be replaced with the name of the meal, "<ingredient_1>" is the name of an ingredient, "<ingredient_proportion>" should be replaced with the proportion of the ingredient needed, "<unit_of_measurement>" should be replaced with the unit of measurement used for the ingredient proportion and "<instruction>" should be replaced with a recipe instruction.
